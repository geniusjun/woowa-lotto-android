# 🎰 Woowa Lotto — Android App (Frontend)

> 이 레포지토리는 우아한테크코스 오픈미션 프로젝트  
> 👉 **[woowa-lotto-spring](https://github.com/geniusjun/woowa-lotto-spring)**  
> 의 **안드로이드 프론트엔드 레포지토리**입니다.

처음으로 도전해본 **코틀린 기반 안드로이드 앱**으로,  
랜덤 로또 구매 / 오늘의 운세 / 랭킹 기능을 제공하는 앱입니다.

---

## ✨ Features

- 🎟 **랜덤 로또 구매** (6 + 보너스 번호 생성)
- 🔮 **오늘의 운세 보기** (1일 1회 제한)
- 🧮 **로또 당첨 여부 UI 표시**
- 🪙 **잔액 관리**
- 🖥️ **백엔드와 연동된 로그인 & 게임 데이터 처리**
- 📱 **전체 화면 Compose UI 구성**

---

## 🛠️ Tech Stack

- **Kotlin**
- **Jetpack Compose**
- **Android Studio**
- **Retrofit**

## 📄 About

3주차 로또 미션을 코틀린으로 리팩토링하다가 “이걸 그냥 앱으로 만들어보면 어떨까?”라는 생각으로 확장한 프로젝트입니다.

처음으로 Android Studio + Jetpack Compose를 사용해 개발하며 모바일 UI/UX 구성, 상태 관리, 화면 구조 설계 등을 처음부터 직접 경험했습니다.
또한 프론트에서 백엔드 api와 연동하는 경험은 처음인데, request와 response의 교환 흐름에 대한 이해도가 많이 올라갔습니다.
